# MobileRobotics2019
Solutions for Intro to Mobile Robotics 2019 (http://ais.informatik.uni-freiburg.de/teaching/ss19/robotics/)opics 
Topics include:
- Motion Models
- Sensor Models
- Localization:
  - Bayes Filter
  - Monte Carlo Particle Filter
  - Kalman Filter
  - Extended Kalman Filter
- Mapping with known poses
- SLAM (fastSLAM)
